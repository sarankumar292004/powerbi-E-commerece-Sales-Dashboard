# E-Commerce Sales Analytics Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze E-Commerce sales performance.

## Key Metrics
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Average Order Value
- Profit Margin %

## Dashboard Features
- Sales by Category Analysis
- Sales by Region Analysis
- Monthly Sales Trend
- Interactive Category Slicer
- Interactive Region Slicer

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Insights
- Furniture generated the highest sales.
- Central region contributed the highest revenue.
- Average Order Value is approximately 308.
- Profit Margin is approximately 12.42%.
